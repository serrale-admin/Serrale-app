# Serrale Home design reference

`serrale-home-reference.png` is the canonical visual reference for the Home screen refresh.

Implementation rules:

- Recreate layout, spacing, card hierarchy, icon treatment, banner composition, and tab emphasis with native Expo components.
- Do not render the full reference image inside the application.
- Keep copy, provider data, navigation, filters, and contact actions native and accessible.
- `../home-trust-banner.png` is the decorative shield-and-Addis artwork extracted from the reference for the native Home banner.
